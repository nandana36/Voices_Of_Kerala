# 🌴 Voices of Kerala

Malayalam Dialect Translator - Translating food words across Kerala's three regions.

## Features
- 🎤 Voice input with speech recognition
- ⌨️ Text input
- 🔊 Text-to-speech for regional pronunciations
- 🔑 API access for developers
- 👨‍💼 Admin panel for word management

## Regions
- 🟡 South Kerala (Trivandrum)
- 🟢 Central Kerala (Kochi)
- 🔵 North Kerala (Calicut)

## Tech Stack
- Streamlit (UI)
- Sarvam AI (Speech Recognition & TTS)
- Google Gemini (AI Translation)

## Deployment
Deployed on Streamlit Cloud
